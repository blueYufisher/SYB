社区简介
发布时间：2017-09-01
       “播种青春创业希望、催生创新创业梦想、汇集多方创业资源、引领校园创业新风！”，为搭建我校学生创业实践平台，增强学生创业意识，培养学生创业能力，广东工业大学大学生创业（孵化）基地于2009年10月孕育而生。

        基地作为学校学生的校内创业实践平台，为在校创业学生提供“场地免费、网络免费、设施免费、培训免费、指导免费、服务免费”的“六免”创业服务。2011年9月，基地成立“南极星创客联盟”学生创业社团组织，充分发挥学生的自我管理、教育与服务，为创业团队提供人员招聘、活动场地申请、对接落实各级创业优惠政策等服务，开展了广工大创业节、创业集市、创业沙龙、校园创业新星评选、走访广工大 创业者、基地开放日等一系列的校园创业文化活动，营造了积极向上的校园创业氛围。

　　基地积极借助以创业校友为主体的社会资源，以“先创业带动后创业、先创业风投后创业、后创业补充先创业、先后创业合作”的“四创合作”模式协同推动校园创业，强化创业校友对在校学生创业的指导与引领作用。2014年，基地引入创业校友组建的基地管理服务公司，协助学校市场化运营管理基地，后成立了广州力燎企业管理有限责任公司，借助校友的经验、资源、人脉等对基地进行专业化管理，对每个入驻基地项目给予“一块场地、一个企业导师、一个校内导师、一个基金池、一个推广平台、一个服务团队”的“六个一”创业联动帮扶服务。与此同时，基地还与学校协同创新平台开展全方位的孵化合作，充分利用协同创新平台的创业资源，深度孵化学生创业项目。

　　目前，基地在孵学生创业项目108个，其中已注册公司63个，拟注册公司8个，带动着1500名参与创业实践的学生活跃在大学校园。基地出孵项目104个，其中，推荐入驻政府科技创业园项目4个，入驻政府创业孵化苗圃项目12个，累计获校级以上创新创业实践项目127项，带动就业1000余人，实现了创业带动就业的倍增效应。基地的发展得到了省市政府部门的充分肯定，2011年、2014年、2015年、2016年分别被评为“广州市创业（孵化）示范基地”、“省级创业带动就业孵化基地”、“大学生创新创业教育示范学校”、教育部首批 “创新创业典型经验高校”、人力资源社会保障部“全国创业孵化示范基地”。
